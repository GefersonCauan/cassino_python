# 🎰 Cassino Python

Simulador de cassino com múltiplas modalidades de jogos, gerenciamento de saldo virtual e sistema de apostas.

---

## 🎮 Jogos disponíveis

- 🎲 Múltiplas modalidades na pasta `jogos/`
- 💵 Sistema de saldo virtual com depósito e saque
- 🏆 Controle de ganhos e perdas por sessão

---

## 🛠️ Tecnologias utilizadas

![Python](https://img.shields.io/badge/Python-3776AB?style=flat&logo=python&logoColor=white)

---

## ▶️ Como executar

**Pré-requisitos:** Python 3.x instalado

```bash
# Clone o repositório
git clone https://github.com/GefersonCauan/cassino_python.git

# Entre na pasta
cd cassino_python

# Execute o jogo
python main.py
```

---

## 📁 Estrutura do projeto

```
cassino_python/
├── main.py       # Ponto de entrada principal
├── saldo.py      # Gerenciamento de saldo virtual
├── utils.py      # Funções utilitárias
├── jogos/        # Módulos de cada jogo
└── data/         # Dados e configurações
```
